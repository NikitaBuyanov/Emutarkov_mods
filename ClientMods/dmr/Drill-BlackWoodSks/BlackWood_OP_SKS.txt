How to install Black Wood OP SKS v1.0 by Drill

You will be replacing a file in your game folder. I highly suggest you make a copy of the file that the one in this zip will be replacing so that you can revert back to the original.


-Black Wood OP SKS stock: stock_sks_molot_op_sks_std.bundle

Copy and paste this file into "\EscapeFromTarkov_Data\StreamingAssets\Windows\assets\content\items\mods\stocks"
